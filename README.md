# pythonsimples

Autor: marcelo marcos

Pequeno repositório de exemplo em Python.

## Estrutura

- `exemplo.py` - exemplo simples em Python
- `requirements.txt` - dependências (se houver)

## Uso

1. Criar e ativar um ambiente virtual (Windows PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Instalar dependências (se necessário):

```powershell
pip install -r requirements.txt
```

3. Executar o exemplo:

```powershell
python exemplo.py
```

## Observações

O diretório `.venv/` está listado no `.gitignore` para evitar versionar a virtualenv.
